◆ hill_break()

std::string hill_break ( std::string  ciphertext)

Выполняет криптоанализ шифра Хилла полным перебором 2×2 матриц

Перебирает все 157 248 обратимых матриц 2×2 по модулю 26 и выбирает ту, при расшифровании которой текст получил наивысшую триграммную оценку. Найденная матрица выводится в std::cout. Работает только при размере блока 2.

Аргументы
ciphertextШифртекст для атаки
Возвращает
Наиболее вероятный открытый текст; хвостовые символы X удалены
Исключения
std::invalid_argumentЕсли шифртекст слишком короткий

std::vector< int >  key_values,
int  n

Зашифровывает текст шифром Хилла (блочный линейный шифр)

Текст делится на блоки длины n; каждый блок умножается на матрицу ключа n×n по модулю 26. Последний неполный блок дополняется буквой X.

Аргументы
plaintextОткрытый текст (английские буквы, регистр не важен)
key_valuesЭлементы матрицы ключа в строчно-мажорном порядке (ровно n*n значений)
nПорядок квадратной матрицы ключа (он же размер блока, от 1 до 8)
Возвращает
Шифртекст в верхнем регистре
Исключения
std::invalid_argumentЕсли матрица не является обратимой по модулю 26, размер неверный, или текст пуст

Возвращает

Исключения
std::invalid_argumentЕсли шифртекст слишком короткий

◆ recurrent_hill_encrypt()

std::string recurrent_hill_encrypt ( std::string  plaintext,
std::vector< std::vector< int > >  keys_values,
int  n 
)

Зашифровывает текст рекуррентным шифром Хилла

Пользователь задаёт m матриц K_1, …, K_m. Блок i шифруется матрицей K_{((i-1) mod m)+1}, т.е. матрицы циклически чередуются.

Аргументы
plaintextОткрытый текст
keys_valuesНабор матриц: keys_values[j] содержит n*n элементов матрицы K_{j+1} (строчно-мажорный порядок)
nПорядок каждой из матриц (он же размер блока, одинаковый для всех)
Возвращает
Шифртекст в верхнем регистре
Исключения
std::invalid_argumentЕсли набор матриц пуст или матрицы некорректны

int  restarts = 60

Выполняет криптоанализ шифра простой замены (дешифрование без ключа)

Использует восхождение к вершине (hill-climbing) по оценке триграмм. Найденный шифралфавит и оценка выводятся в std::cout. Точность растёт с длиной шифртекста (рекомендуется не менее 100 букв).

Аргументы
ciphertextШифртекст для атаки
restartsЧисло случайных перезапусков алгоритма (по умолчанию 60)
Возвращает
Наиболее вероятный открытый текст
Исключения
std::invalid_argumentЕсли шифртекст слишком короткий для анализа
